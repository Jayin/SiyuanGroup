# siyuan

Server side of the SiyuanGroup project, based on Node


## Installation

```shell
cd node
npm install -d
```

```shell
cd config
cd sql
mysql -u <dbuser> -p
<dbpwd>
source init.sql
source <*.sql>
...
cd ..
```


## Running

```shell
npm start
```


## Testing

```shell
npm install -g mocha
```

```shell
npm test
```
